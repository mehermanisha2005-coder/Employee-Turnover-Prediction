# Employee Turnover Prediction

## Project Overview

Employee turnover is a critical challenge for organizations as it impacts productivity, recruitment costs, and workforce stability. This project aims to predict employee attrition using Machine Learning techniques, enabling organizations to identify employees at risk of leaving and take proactive retention measures.

Using employee-related features, a predictive model was developed to analyze turnover patterns and classify whether an employee is likely to leave the organization.

---

## Objectives

* Analyze employee data to identify factors influencing turnover.
* Perform data preprocessing and feature engineering.
* Build and evaluate predictive Machine Learning models.
* Compare Logistic Regression models with L1 (Lasso) and L2 (Ridge) regularization.
* Generate insights to support HR decision-making.

---

## Dataset Information

* Total Records: 1,350 Employees
* Features: 15+ Employee Attributes
* Target Variable: Employee Turnover (Attrition)

The dataset contains employee-related information used to predict attrition behavior and workforce trends.
